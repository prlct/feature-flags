# Email sequence webhooks

- Start - `POST: https://api.growthflags.com/sequences/webhook/start/:eventKey`
- Stop - `POST: https://api.growthflags.com/sequences/webhook/stop/:eventKey`

### Params
 - `email`
 - `eventKey`
 - `firstName` - optional
 - `lastName` - optional
